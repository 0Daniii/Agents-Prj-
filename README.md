# Automated Research & Summarization Team

A **multi-agent intelligent system** implemented in Python that automatically:
- Researches a topic
- Extracts key information
- Summarizes findings
- Validates coherence

## Agents
- ResearchAgent: Collects raw information
- AnalysisAgent: Extracts key points
- SummaryAgent: Generates a clean summary
- ReviewAgent: Validates and improves output

## Tech
- Python 3.10+
- No external APIs required
- Fully local & deterministic

## Run
```bash
python main.py
